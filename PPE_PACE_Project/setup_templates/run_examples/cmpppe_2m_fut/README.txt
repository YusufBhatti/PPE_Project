Specificities of this template:
-------------------------------
Template for the simplification simulations in future SST conditions
(simplifications derived form the 2M CMPPPE (2 moment CMP scheme, cloud microphsyics perturbed parameter ensemble)
The simplifications to be used need to be set with @lcmpsimpl_prescr@ (general turn on simplifications), @ncmpsimpl_prescr_rime@, @ncmpsimpl_prescr_icnucl@, 
@ncmpsimpl_prescr_subfis@, @ncmpsimpl_prescr_sci@, and @ncmpsimpl_prescr_icaccr@ (values 0 for no simplification, 1 for constant, 2 for climatology)
